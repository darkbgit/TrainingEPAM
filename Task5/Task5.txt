Система отображения статистики продаж

Web приложение основанное на ASP.NET MVC, в качестве наполнялки базы может использоваться приложение из предедущей задачи.

- приложение читает данные о продажах из базы и отображает их в виде, удобном для анализаю Должны быть созданы контроллеры, представления и модели представлений для каждой сущностию Должны быть реализована возможность фильтрации как минимум по двум свойствам для каждой из сущностей.
- в системе должна присутствовать аутентификация пользователей. Должны существовать две роли - простой пользователь и администраторю Пользователь может только просматривать данные, не имеет право ничего редактировать. Администратор может изменять записи в базе данных.
- интерфейс для вывода информации придумать и реализовать самостоятельно (приветствуется использование JavaScript фреймворков). Вывод информации в таблицы должен выполняться с помощью асинхронных AJAX запросов.
- обеспечить валидацию запросов на стороне клиента и сервера (проверка макс. длины, типов данных, формата даты)
- желательно построение диаграмм на основании имеющихся данных, оси - произвольные. Виды диаграмм выбрать самостоятельно.
